# 🎓 Student Hub Management System

## 📖 Overview

Student Hub Management System is a responsive web application developed using HTML, CSS, and JavaScript. It provides different dashboards for Admin, Students, Teachers, and Parents to manage academic information efficiently.

---

## ✨ Features

### 👨‍💼 Admin
- Dashboard
- Student Management
- Teacher Management
- Attendance Overview
- Results Management
- Notices

### 👨‍🎓 Student
- Dashboard
- View Attendance
- View Results
- View Timetable
- View Notices

### 👨‍🏫 Teacher
- Dashboard
- Class Schedule
- Attendance Management
- Results

### 👨‍👩‍👦 Parent
- Dashboard
- Child Progress
- Attendance
- Results
- Notices

---

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Objective

The main objective of this project is to provide a simple and user-friendly Student Management System that improves communication between administrators, teachers, students, and parents.
